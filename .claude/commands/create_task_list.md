# Task List 작성 안내


## Task List 작성 위치
```
project-root/
└── tasks/
    ├── feature-group1 name/
    │   ├── task1.md
    │   ├── task2.md
    │   └── ...
    ├── feature-group2 name/
    │   ├── task1.md
    │   ├── task2.md
    │   └── ...
```
- 각 Task List는 `tasks/<feature-group name>/` 아래에 위치
- Feature Group 폴더가 없으면 새로 생성

## Task List 포함 항목
### 문서 상단 부분
- **Feature Group 이름**
- **Task List 제목**
- **최초 작성 시각**
