React 18이 공식 출시되면서 많은 새로운 기능들이 추가되었습니다. 특히 Concurrent Rendering, Automatic Batching, Suspense의 개선사항들이 주목할 만합니다. ## 주요 변경사항 1. **Concurrent Rendering**: 렌더링 프로세스를 중단하고 재개할 수 있게 되어 사용자 경험이 크게 향상되었습니다. 2. **Automatic Batching**: React 18에서는 Promise, setTimeout, native event handler 등에서도 자동으로 배칭이 이루어집니다. 3. **Suspense SSR**: 서버 사이드 렌더링에서 Suspense를 사용할 수 있게 되어 로딩 성능이 개선되었습니다. 이러한 변경사항들은 개발자 경험과 사용자 경험 모두를 크게 향상시킬 것으로 예상됩니다.
